# Quebra cabeça de 8 peças

Resolve um quebra cabeça de 8 peças usando 3 métodos:
 - Manhattan
 - Hamming
 - Heuristc

### 📋 Pré-requisitos

Precisa instalar o MatLab.

## ⚙️ Executando os testes

Execute os arquivos:
 - Ex1_Heuristc.m
 - Ex1_Manhattan.m
 - Ex_Hamming.m
